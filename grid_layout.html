Kesimpulan skenario 1 : Penggunaan CSS Grid memungkinkan penataan item-item dalam tempat_grid menjadi lebih fleksibel.

Kesimpulan skenario 2 : Kode program ini merupakan contoh penggunaan CSS Grid untuk membuat tata letak grid sederhana dengan tiga kolom yang disusun secara inline, serta menampilkan isi grid dengan nomor 1 sampai 9.

Kesimpulan skenario 3 : Grid gap digunakan untuk mengatur jarak antar item pada tempat_grid. Column gap mengatur jarak antar kolom, sedangkan row gap mengatur jarak antar baris.

Kesimpulan skenario 4 : Column gap dan row gap dapat diatur secara bersamaan dengan menggunakan properti grid-gap.
